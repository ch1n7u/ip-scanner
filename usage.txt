1. Basic scan without ports:

    python scan_ips.py ips.txt -o report.pdf

2. Scan with specific ports:

    python scan_ips.py ips.txt -o report.json -p 80,443,8080

3. Scan with port range:

    python scan_ips.py ips.txt -o report.pdf -p 20-25

4. With custom thread count:

    python scan_ips.py ips.txt -o report.json -p 22,80,443 -t 8
